Warp
================================================================================

Warp to the another world


Requirements
--------------------------------------------------------------------------------

- [Peco](https://github.com/peco/peco)


Installation
--------------------------------------------------------------------------------

Download to your executable path.

```sh
$ cd $HOME/bin
$ curl -LO https://raw.githubusercontent.com/Tomohiro/warp/master/warp
$ chmod +x ./warp
```

Usage
--------------------------------------------------------------------------------

```sh
$ warp
```
